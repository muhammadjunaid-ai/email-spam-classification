# Email Spam Classification

## Project Description
This project is a Machine Learning web application that classifies email or SMS messages as **Spam** or **Not Spam**. The model uses **Natural Language Processing (NLP)** techniques such as **TF-IDF vectorization** and a **Naive Bayes classifier** to analyze the text and make predictions. The application is built with **Python, Scikit-learn, and Streamlit**, and it allows users to enter a message and instantly see whether it is spam or legitimate.
